# Dropship System with Admin Panel

A complete dropship management system with separate admin panel and customer website, optimized for Vercel deployment.

## Features

### Admin Panel
- Dashboard with statistics
- Product management (CRUD)
- Order management
- Customer management
- Supplier management
- Sales reports

### Customer Website
- Product catalog with filters
- Shopping cart
- Checkout process
- User accounts
- Order tracking

## Deployment on Vercel

### Method 1: Deploy via Vercel CLI
```bash
# Install Vercel CLI
npm install -g vercel

# Login to Vercel
vercel login

# Deploy
vercel --prod
